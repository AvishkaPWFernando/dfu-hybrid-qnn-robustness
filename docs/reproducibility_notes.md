# Reproducibility notes

The experiments were run in Google Colab GPU environments.

Reported environment details from the manuscript include:

- Python 3.12-based Colab runtime
- PyTorch
- Torchvision
- PennyLane
- NumPy
- scikit-learn
- Matplotlib

The fixed random seed was set to 42 where supported.

The notebooks include the model definitions, training setup, evaluation workflow and reported outputs for the five model tracks.

Important limitations:

- The raw dataset is not redistributed.
- Some results may vary slightly due to Colab runtime, GPU, package and random-seed behaviour.
